Flashcard Carousel

OSI has 7 layers: Physical, Data Link, Network, Transport, Session, Presentation, Application
CIDR: /24 subnet = 255.255.255.0, 256 IPs, 254 usable
802.1Q is the trunking protocol tagging VLANs in Ethernet frames
OSPF uses cost metric based on bandwidth; EIGRP uses bandwidth and delay
ACLs filter traffic. Standard ACLs use source IP only. Extended ACLs can filter by protocol, source & destination IPs and ports.
